# Django To-Do List App

### A fully functional To-Do List web application built with Django, featuring user authentication and complete CRUD operations. This project is designed to help users manage their daily tasks efficiently.

* User Registration & Authentication (Login / Logout)
* Create, Read, Update, and Delete (CRUD) Tasks
* Task Search Functionality
* Mark Tasks as Complete / Incomplete
* Organized task management per user
* Clean and minimal UI

## Technologies Used
* Python
* Django
* HTML / CSS
* SQLite
